# IonicTodoApp/InStachGram
Todo App developed with ionic framework for the Movember !

## Getting Started
```
git clone https://github.com/theben67/IonicTodoApp.git
```
```
cd IonicTodoApp
```
```
ionic serve //(select IP) For watch app in your browser
```
